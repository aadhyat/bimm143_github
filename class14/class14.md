## DESeq Analysis

### Setting up the DESeq object

``` r
library(DESeq2)
```

Build the required DESeqDataSet object for DESeq analysis:

``` r
dds <- DESeqDataSetFromMatrix(countData = countData,
                             colData = colData,
                             design = ~condition)
```

Improve the plot with color and lines to highlight significant changes.

``` r
my_cols <- rep("gray", nrow(res))
my_cols[abs(res$log2FoldChange) > 2 ] <- "blue"
my_cols[res$padj>=0.01] <- "gray"
```

``` r
ggplot(res) +
      aes(x=log2FoldChange,
          y=-log(padj)) +
      geom_point(col=my_cols) +
      geom_vline(xintercept = c(-2,+2), col="red") +
      geom_hline(yintercept = -log(0.01), col="red") +
      labs(x = "Log2(Fold Change)",
```
